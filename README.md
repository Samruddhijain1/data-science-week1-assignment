# Data Science Week 1 Assignment – Flipkart Sales Analysis

## Overview

This project was completed as part of the Data Science Internship Week 1 Assignment. The objective was to perform data cleaning, sales analysis, business insight generation, data visualization, and dashboard creation using an e-commerce sales dataset.

## Objectives

* Clean and validate raw sales data.
* Handle missing values and duplicate records.
* Perform exploratory data analysis (EDA).
* Calculate key business KPIs.
* Generate business insights and recommendations.
* Create professional visualizations.
* Design a dashboard summarizing key findings.

## Dataset

**Dataset Name:** Flipkart Sales Dataset

The dataset contains transactional sales information including:

* Order Details
* Product Information
* Customer Demographics
* Revenue Data
* Delivery Status
* Sales Dates
* Product Categories

## Task 2: Sales Data Analysis

### Key Performance Indicators (KPIs)

| KPI                 | Value             |
| ------------------- | ----------------- |
| Total Revenue       | GHS 25.42 Million |
| Total Orders        | 113,000           |
| Average Order Value | GHS 224.97        |
| Total Units Sold    | 602,834           |
| Top Category        | Health and Beauty |

### Major Findings

1. Health and Beauty generated the highest revenue among all product categories.
2. January recorded the highest monthly sales performance.
3. Zone 3 generated the highest revenue.
4. Male customers contributed slightly higher revenue than female customers.
5. Customers aged 26–35 placed the highest number of orders.
6. Shipped-from-Abroad deliveries recorded the highest return rate.
7. Revenue showed significant growth over time, with 2020 being the best-performing year.

* Increase focus on Health and Beauty products through targeted promotions.
* Investigate causes of product returns to reduce return rates.
* Replicate successful strategies that led to sales peaks.
* Focus marketing efforts on the 26–35 age segment.
* Maintain inventory levels for top-selling products.
